Broadsheet/Analysis

To view the broadsheet analysis of the first half or end-of-term result, follow these steps:

  1. Select the desired session year from the available options.
  2. Choose the term (first, second, or third) for which you want to view the results.
  3. Select the result type (first half or end of term).
  4. Select the class for which you want to analyze the result.
  5. Click the "Submit" button to generate the broadsheet analysis.

After submitting the required information, the webpage will display the following charts (Analysis) and table (Broadsheet):

Charts (Analysis):

  1. Doughnut Chart (SUBJECT BASED: Analysis of Number of Grades Below): This chart shows the number of grades below C for each subject offered by the class.
  2. Pie Chart (ANALYSIS OF CLASS AVERAGE GRADE COUNT): This chart displays the number of grades obtained in total, calculated from the average score of each student.
  3. Stacked Bar Chart (ANALYSIS OF SUBJECTS BASED GRADES): This chart represents the total number of each grade achieved by students in each subject.
  4. Double Bar Chart (Previous vs. Current Term Performance - Subjects): This chart compares the performance of each subject between the previous and current term.
  5. Double Bar Chart (Previous vs. Current Term Performance - Students): This chart compares the performance of each student between the previous and current term.

Table (Broadsheet):

The table displays the students in the selected class, along with their corresponding average score, average grade, and total score for each subject.

Notes:

  • Display: the content of the web page will adjust to the device used as much as possible but cannot guarentee of 100% fit, use chrome or edge with zoom between 75% to 85% for optimized display
  • View: The generated result will be displayed on the webpage, allowing you to scroll through and review it.
  • Print: There is an option to print the result directly from the webpage.
  • Download as PDF: You can choose to download the result as a PDF file, which can be saved and accessed offline.

Student Based Result

To check the result for a student, follow these steps:

  1. Select the desired session year from the available options.
  2. Choose the term (first, second, or third) for which you want to view the results.
  3. Select the result type (first half or end of term).
  4. Enter the student ID into the provided input field.
  5. Click the "Submit" button to generate the complete result.

After submitting the required information, the webpage will display the student's complete result. Users have multiple options for interacting with the result:

  • View: The generated result will be displayed on the webpage, allowing you to scroll through and review it.
  • Print: There is an option to print the result directly from the webpage.
  • Download as PDF: You can choose to download the result as a PDF file, which can be saved and accessed offline.

In cases where the result does not exist for the provided input, the webpage will display a message informing the user about the absence of the result.

Subject based result

To check the result for a subject, follow these steps:

  1. Select the desired session year from the available options.
  2. Choose the term (first, second, or third) for which you want to view the results.
  3. Specify the result type (first half or end of term).
  4. Select the subject from the options list.
  5. Click the "Generate" button to generate the result.

After submitting the required information, the webpage will display the detailed result for the selected subject. If the selected subject is a combined subject, the result for the sub-subjects will be displayed first, followed by the combined result at the bottom.

Users have the option to print the result directly from the webpage.
